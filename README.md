# Real-Estate-App
A fullstack **Real Estate App** built with **Next.js** for the frontend and **Node.js, REST APIs** for the backend.  
This application allows users to browse properties, view property details, and contact property owners. Admins can manage listings via a backend dashboard.

---

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/jatinm79/real-estate-app.git

cd real-estate-git

```
###  2. Install dependencies

Backend
```bash
cd backend
npm install

```
Frontend
```bash
cd frontend
npm install
```

## Running the project

### Start the Backend
```bash
cd backend
npm run dev
```
The backend will start at http://localhost:4000

### Start the Frontend
```bash
cd frontend
npm run dev
```
The frontend will start at http://localhost:3000
